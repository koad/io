Identity

No cached sovereign profile.

Cached on each Tier 1/2 connection from /api/sovereign-profile. Public data only — the extension never holds private keys. Used to render the offline fallback view in the side panel.

Outbound queue

0 queued · 0 dead-lettered

Writes (like injectContext) made while offline (Tier 3) are buffered to chrome.storage.local and flushed when Tier 1/2 returns. Dead-lettered entries failed MAX_ATTEMPTS times and are kept for inspection only.

Session token (MCP)

(none)

UUID generated at extension startup, sent as Authorization: Bearer <token> to the daemon. Cleared on browser close. Per SPEC-196 §4 — the extension never holds private keys; this is the only credential surface.
