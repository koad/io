# koad-io-core (placeholder)

**Status:** Empty directory — not a functional package.

This directory exists as a placeholder. It contains no `package.js` and no source files.

Do not confuse this with `core/`, which is the actual `koad:io-core` package — the foundation that all koad:io Meteor packages depend on.

## What You're Looking For

If you want the core package:

```bash
meteor add koad:io-core
```

That is provided by the `core/` directory in this packages tree. See `core/README.md`.

If you want everything:

```bash
meteor add koad:io
```

That is provided by the `koad-io/` directory. See `koad-io/README.md`.
